# JXFinance

抖音店铺达人结算工具 MVP。

## 功能

- 上传抖店订单明细表和资金流水明细表，生成结算输出表。
- 可选上传商品成本明细表和运费险明细表。
- 输出包含订单明细融合表、达人结算汇总表、资金流水总览、资金流水场景汇总、资金流水月份汇总、资金流水明细核对。
- 默认快递成本为每个包裹 2.4 元。

## 商品成本明细表字段

商品成本明细表为人工维护表，MVP 阶段字段名固定：

- `商家编码`
- `产品成本`

多余字段会被忽略，缺少固定字段会停止生成并提示错误。
