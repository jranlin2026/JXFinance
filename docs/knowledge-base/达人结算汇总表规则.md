# 达人结算汇总表规则

## 一句话结论

达人结算汇总表是最终给达人结算端使用的利润汇总表：以订单表为主归属口径，按 `订单月份 + 达人昵称 + 达人ID` 聚合实付订单金额、订单数、快递、运费险、结算到账、产品成本，并计算成本总额、毛利润和销售额毛利率。

本规则已用 `风华服饰时尚小铺` 2026 年 5 月订单表、资金流水表、运费险表、商品单价表验证通过。

## 输入表

```text
订单表：主表，用于订单月份、达人归属、实付金额、实付订单数、快递包裹数、商品数量、商家编码。
资金流水表：用于计算订单实际结算到账金额。
运费险表：可选，用于计算已扣减的运费险费用；缺失时按 0 费用处理。
商品单价表：可选，用于按商家编码取得产品单件成本；缺失时按 0 成本处理。
```

## 输出字段

字段顺序固定如下：

```text
订单月份
达人昵称
达人ID
实付订单金额
实付订单数
快递包裹数
快递费用
运费险费用
结算到账金额
产品成本
成本总额
毛利润
销售额毛利率
```

## 分组口径

```text
分组键 = 订单月份 + 达人昵称 + 达人ID
订单月份 = 订单提交时间的月份，格式 YYYY-MM
```

达人归属：

```text
达人昵称为空 -> 达人昵称 = 商品卡流量
达人昵称非空 -> 使用订单表原达人昵称
达人昵称为空时，达人ID 留空
```

注意：订单数据表格可能包含多个月份，必须先根据 `订单提交时间` 取订单月份，再按月份分别汇总，不能把不同月份的同一达人合并到一行。

## 实付订单金额

只统计 `支付完成时间` 非空的订单行。

```text
实付订单金额 =
订单应付金额
+ 平台实际承担优惠金额
+ 商家实际承担优惠金额
+ 达人实际承担优惠金额
```

`支付完成时间` 为空代表用户拍下但未付款，不参与实付订单金额计算。

## 实付订单数

只统计 `支付完成时间` 非空的实际付款子订单数。

```text
实付订单数 = 支付完成时间非空订单中的去重子订单编号数
```

## 快递包裹数

快递包裹数使用已确认的 `快递包裹数计算规则`。

```text
只统计发货时间非空的订单
按 订单月份 + 达人昵称 + 达人ID + 主订单编号 + 发货时间 去重
每个唯一组合算 1 个快递包裹
```

一个主订单下有多个子订单时，如果发货时间相同，算同一个包裹；如果发货时间不同，按不同发货时间分别算包裹。

## 快递费用

当前默认每个快递包裹成本为 2.4 元。

```text
快递费用 = 快递包裹数 * 2.4
```

后续如果每个店铺或月份快递单价不同，应把 `2.4` 做成可配置参数。

## 运费险费用

运费险费用来自运费险表，只统计 `保费状态 = 已扣减` 的记录。

```text
保费状态 = 已扣减 -> 计入运费险费用
保费状态 = 不扣减 -> 不计入运费险费用
```

归属方式：

```text
1. 读取运费险表的订单编号。
2. 只匹配订单表的子订单编号。
3. 不使用主订单编号兜底匹配运费险。
4. 匹配成功后，按订单表中的订单月份、达人昵称、达人ID 归属。
5. 匹配不到订单表子订单编号的运费险记录，不归属到任何达人，并作为异常记录。
```

注意：运费险只按子订单编号归属。不能使用主订单编号兜底，否则同一主订单下多子订单会导致费用偏高。

## 结算到账金额

结算到账金额来自资金流水表，只看资金流水实际发生的入账和出账，不看订单状态。

```text
结算到账金额 = 资金流水入账金额 - 资金流水出账金额
```

资金流水匹配规则：

```text
1. 如果资金流水有子订单号，只能匹配订单表的子订单编号。
2. 只有当资金流水子订单号为空时，才允许用资金流水订单号匹配订单表主订单编号。
3. 不允许用某个子订单流水覆盖同一主订单下的其他子订单。
4. 正数和负数都保留，负数是正常情况。
5. 匹配不到订单表的资金流水，不强行分摊到达人，并作为异常记录。
```

这个规则用于避免产品成本和结算金额被主订单维度错误放大。

## 产品成本

产品成本只认商品单价表，不从订单表或其他字段反推。

匹配方式：

```text
订单表.商家编码 = 商品单价表.商家编码
产品单件成本 = 商品单价表.产品单价
```

成本计算条件：

```text
只有订单行的结算到账金额 > 5 元，才计算产品成本。
结算到账金额 <= 5 元，或没有结算到账金额时，产品成本为空，不参与汇总。
```

计算公式：

```text
产品成本 = 产品单件成本 * 商品数量
```

最终达人结算汇总表中的 `产品成本` 是按分组键汇总后的产品成本合计。

## 成本总额

```text
成本总额 = 快递费用 + 运费险费用 + 产品成本
```

## 毛利润

```text
毛利润 = 结算到账金额 - 成本总额
```

这里的结算到账金额可以为正，也可以为负；负数、退款、扣款都按资金流水实际结果参与计算。

## 销售额毛利率

当前销售额口径使用 `实付订单金额`。

```text
销售额毛利率 = 毛利润 / 实付订单金额
```

如果 `实付订单金额 = 0`，销售额毛利率留空，避免除以 0。

## 使用提醒

```text
达人结算汇总表是最终利润表，不是订单行级明细表。
订单明细融合表用于追溯单行订单来源，达人结算汇总表用于最终结算口径。
涉及运费险、资金流水、产品成本时，应分别按各自源表的去重和匹配规则汇总，不能直接把明细展示字段简单求和。
```
