## 当前主线口径

订单表和资金流水表是结算输出表的必填输入。商品成本明细表和运费险明细表是可选输入；缺失时分别按 0 成本、0 费用处理。

订单表是达人利润统计的主表之一。按达人维度统计利润前，需要先形成两类基础表：

```text
1. 订单明细融合表：订单行级底表，用于追溯每个订单行的到账、运费险和产品成本。
2. 达人实付订单汇总表：达人维度基础汇总表，用于归集实付金额、订单数、包裹数、运费险和结算到账。
3. 达人结算汇总表：最终利润汇总表，用于输出成本总额、毛利润和销售额毛利率。
```

标准交付是一个 `结算输出表.xlsx` 工作簿，里面包含 `订单明细融合表` 和 `达人结算汇总表` 两个 sheet。订单明细融合表从订单表出发，挂接资金流水、运费险和商品单价表；达人汇总表再基于同一套口径按达人聚合。
